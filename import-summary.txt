Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => space\src\main\AndroidManifest.xml
* assets\ => space\src\main\assets\
* libs\armeabi-v7a\libmain.so => space\src\main\jniLibs\armeabi-v7a\libmain.so
* libs\armeabi-v7a\libmono.so => space\src\main\jniLibs\armeabi-v7a\libmono.so
* libs\armeabi-v7a\libsqlite3.so => space\src\main\jniLibs\armeabi-v7a\libsqlite3.so
* libs\armeabi-v7a\libunity.so => space\src\main\jniLibs\armeabi-v7a\libunity.so
* libs\unity-classes.jar => space\libs\unity-classes.jar
* libs\x86\libmain.so => space\src\main\jniLibs\x86\libmain.so
* libs\x86\libmono.so => space\src\main\jniLibs\x86\libmono.so
* libs\x86\libsqlite3.so => space\src\main\jniLibs\x86\libsqlite3.so
* libs\x86\libunity.so => space\src\main\jniLibs\x86\libunity.so
* res\ => space\src\main\res\
* src\ => space\src\main\java\
